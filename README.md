# ⭐ Loopstudios Landing Page

Landing page responsive inspirada en el reto de Frontend Mentor **"Loopstudios Landing Page"**.  
El objetivo fue desarrollar una interfaz moderna y adaptable, implementando un menú de navegación móvil interactivo, diseño mobile-first, buenas prácticas de accesibilidad y una estructura semántica utilizando HTML, Tailwind CSS y JavaScript.

---

## 📌 Descripción

Este proyecto consiste en una landing page para una empresa enfocada en experiencias de realidad virtual.

Se desarrolló con un enfoque mobile-first y utiliza imágenes responsivas mediante el elemento `<picture>` para optimizar la experiencia en distintos tamaños de pantalla.

Además, se implementó un menú de navegación para dispositivos móviles con JavaScript, permitiendo mostrar y ocultar el menú de forma dinámica, bloquear el desplazamiento de la página mientras permanece abierto y actualizar atributos ARIA para mejorar la accesibilidad.

La interfaz fue construida utilizando utilidades de Tailwind CSS, priorizando una estructura limpia, componentes reutilizables y un código organizado.
